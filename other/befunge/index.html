What is Befunge?

Befunge is an interpreted, two-dimensional, stack-based programming language originally created in 1993 by Chris Pressey.

In Befunge the program is contained in a grid of cells. Each cell contains a single command, expressed by a single character. As the program counter moves to an adjacent cell based the direction it is already moving. Commands that change directions: >, <, ^, v, and ? When the program counter reaches the sides, top, or bottom, it wraps around.

While the program is running values may be pushed onto and popped off the stack. The stack is simply a list of numbers that iskept in memory by Befunge as it runs the program.
