Installing CouchDB on Raspberry Pi

last updated: 2020-04-22

This is really just a condensed version of the install instructions from CouchDB.

This covers installing CouchDB 3.0.0, the latest version available as of this writing.

Install Prerequisite Packages

sudo apt install build-essential pkg-config erlang libicu-dev libcurl4-openssl-dev libmozjs185-dev

Download CouchDB Source Code

wget http://apache.mirrors.pair.com/couchdb/source/3.0.0/apache-couchdb-3.0.0.tar.gz

Unzip the Source

tar xvzf apache-couchdb-3.0.0.tar.gz

Build CouchDB

cd apache-couchdb-3.0.0 
./configure  
make release
